## 데이터 흐름 (재현 단계)

```bash
# 1. .hwp → .hwpx (한컴 COM 1회)
python scripts/hwp_to_hwpx.py \
    "samples/rfp_downloaded/[양식] 농식품 분야 「AI 응용제품 신속상용화 지원사업」.hwp" \
    "output/20260531/농식품AI_양식.hwpx"

# 2. 양식 분석 (셀·placeholder·intent 분류)
python scripts/extract_hwpx_form.py \
    "output/20260531/농식품AI_양식.hwpx" \
    "output/20260531/통합양식.form.yaml"

# 3. KB 자동 채움 (회사메타·재무)
python scripts/fill_company_cells.py \
    "output/20260531/통합양식.form.yaml" \
    "kb/company/dabeeo/profile.yaml" \
    "output/20260531/fills_profile.yaml"

python scripts/fill_finance_cells.py \
    "output/20260531/통합양식.form.yaml" \
    "kb/company/dabeeo/finance.yaml" \
    "output/20260531/fills_finance.yaml"

# 4. profile + finance 병합
python -c "
import yaml
f1 = yaml.safe_load(open('output/20260531/fills_profile.yaml', encoding='utf-8'))['fills']
f2 = yaml.safe_load(open('output/20260531/fills_finance.yaml', encoding='utf-8'))['fills']
seen, merged = set(), []
for f in f1 + f2:
    if f['id'] not in seen:
        seen.add(f['id']); merged.append(f)
yaml.dump({'fills': merged}, open('output/20260531/fills_merged.yaml', 'w', encoding='utf-8'), allow_unicode=True, sort_keys=False)
"

# 5. proposal-writer agent — 본체 별지 3 본문 채움 (~15분)
#    비즈니스 결정: 단독 / 타입1 / 28.6억 / Eartheye Plantation
#    → output/20260531/fills_본체별지3_v5.yaml

# 6. merged + v5 통합
python -c "
import yaml
f1 = yaml.safe_load(open('output/20260531/fills_merged.yaml', encoding='utf-8'))['fills']
f2 = yaml.safe_load(open('output/20260531/fills_본체별지3_v5.yaml', encoding='utf-8'))['fills']
seen, merged = set(), []
for f in f1 + f2:
    if f['id'] not in seen:
        seen.add(f['id']); merged.append(f)
yaml.dump({'fills': merged}, open('output/20260531/fills_total_v5.yaml', 'w', encoding='utf-8'), allow_unicode=True, sort_keys=False)
"

# 7. fill_hwpx (XML 결정적 채움)
python scripts/fill_hwpx_form.py \
    "output/20260531/농식품AI_양식.hwpx" \
    "output/20260531/fills_total_v5.yaml" \
    "output/20260531/농식품AI_v5.hwpx"

# 8. 별지 분할 (9개 .hwpx)
python scripts/split_hwpx_by_section.py \
    "output/20260531/농식품AI_v5.hwpx" \
    "output/20260531/통합양식.form.yaml" \
    "output/20260531/별지_v5"

# 9. PDF + 텍스트 (시각·검토)
python scripts/hwpx_to_pdf.py \
    "output/20260531/별지_v5/05_[별지_제3호]_사업계획서.hwpx" \

    "output/20260531/별지3호_v5_hi" 150
```

## 검토 방법론 (이번 라운드 발견)

```bash

# 그 후 .txt grep / sed -n '<from>,<to>p' 로 페이지 검토
```

PNG 는 *레이아웃 검증·overflow 확인*에만 사용.
