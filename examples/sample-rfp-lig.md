# 제안요청서 (RFP) — 무인기 통합 감시정찰 체계 시범사업

> **주의:** 본 문서는 DSI 시스템 테스트용 가공 데이터입니다. 실제 RFP가 아니며, 실제 발주기관·예산·일정과 무관합니다.

## 1. 사업 개요

| 항목 | 내용 |
|---|---|
| 사업명 | 차세대 무인기 통합 감시정찰 체계 시범사업 |
| 발주기관 | (가칭) 방위사업청 무인체계사업단 |
| 사업기간 | 계약일로부터 18개월 |
| 사업예산 | 320억원 (VAT 별도) |
| 사업분류 | 방산 / 무기체계 시범사업 |
| 계약방식 | 협상에 의한 계약 |

## 2. 추진 배경 및 목적

- 접경 지역 24시간 감시 수요 증가 및 인력 의존도 완화 필요
- 기존 유인 감시 자산의 운용 비용 및 인적 위험 부담
- 무인기 기반 자율 감시정찰을 통한 ISR(감시·정찰·정보) 능력 강화
- AI 영상분석 기반 표적 자동 탐지·식별로 지휘 의사결정 시간 단축
- 향후 대량 양산 사업을 위한 시범 체계 구축 및 운용 데이터 확보

## 3. 사업 범위

### 3.1 무인기 플랫폼 (4개 편대, 총 16대)
- 중·고고도 정찰용 무인기 4대 (체공 12시간 이상)
- 중·저고도 전술 무인기 12대 (체공 4시간 이상)
- 모든 기체는 GPS 거부환경 대응 항법 모듈 탑재

### 3.2 탑재 임무장비
- EO/IR 통합 감시 카메라 (주야간 동시 운용)
- 합성개구레이더(SAR) 모듈 (전천후 감시)
- 전자전 정찰 페이로드(ELINT/SIGINT)
- 데이터링크 (Link-K 호환)

### 3.3 지상 통제 시스템 (GCS)
- 다중 기체 동시 운용 (편대 단위 자동 임무 할당)
- AI 영상분석 모듈 (표적 자동 탐지·식별·추적)
- 지휘통제(C2) 통합 인터페이스
- 상황 인식 통합 대시보드

### 3.4 운영 지원
- 18개월 시범운용 기간 중 유지보수
- 운용자 교육 (조종사·분석관·정비사 각 2회)
- 운용 매뉴얼·교범 작성
- 양산 사업 전환을 위한 사업 결과보고서 작성

## 4. 자격 요건

- **방산업체 지정** (방위사업법 제35조)
- 최근 5년 내 무인기 또는 정밀유도무기 체계 양산 실적 1건 이상
- 최근 3년 내 C4ISR 체계 사업 수행 실적 1건 이상
- **국방품질경영시스템(DQMS) 인증** 보유
- **CMMI Level 3 이상** (SW 개발 사업 포함)
- 보안업체 인증 (군사기밀 II급 이상 취급 자격)
- 자본금 500억원 이상
- 컨소시엄 구성 허용 (주관사 + 최대 4개 참여사)

- 지상 통제소 동시 처리 영상 채널 16개 이상
- 사이버 보안: 통신 암호화, 안티-재밍 적용
- 군사기밀 II급 취급 보안 요건 준수
