# 제안요청서 (RFP)

> **주의:** 본 문서는 DSI 시스템 테스트용 가공 데이터입니다. 실제 RFP가 아님.

## 1. 사업 개요

| 항목 | 내용 |
|---|---|
| 사업명 | 가상공항 실내 내비게이션 시스템 구축 |
| 발주기관 | (가칭) 한국가상공항공사 |
| 사업기간 | 계약일로부터 9개월 |
| 사업예산 | 5억 5천만원 (VAT 별도) |
| 사업분류 | 정보화 / 시스템 통합 (SI) |
| 계약방식 | 협상에 의한 계약 |

## 2. 추진 배경 및 목적

- 여객 증가에 따른 공항 내 길찾기 서비스 수요 증가
- 기존 안내 시스템(표지판·키오스크)의 정보 갱신 지연 및 다국어 한계
- 실내 측위 기반 모바일 내비게이션을 통한 여객 편의 향상
- 환승 동선 최적화로 지연 출국 감소 목표

## 3. 사업 범위

### 3.1 시스템 구축
- 실내 측위 인프라 (BLE 비콘 / Wi-Fi RTT) 설계·설치
- 모바일 앱 (iOS·Android) 개발
- 관리자용 콘텐츠 운영 시스템 (CMS)
- 운영 데이터 수집·분석 대시보드

### 3.2 콘텐츠 구축
- 공항 3D 실내 지도 (3개 터미널 × 평균 2개 층)
- 주요 시설물 메타데이터 (게이트·면세점·식음·라운지 등)
- 다국어 콘텐츠 (한국어·영어·중국어·일본어)

### 3.3 운영 지원
- 1년간 유지보수
- 운영자 교육 (2회)
- 콘텐츠 갱신 가이드 문서

## 4. 자격 요건

- 소프트웨어사업자 신고 필요
- 정보보호관리체계(ISMS) 인증 보유
- 최근 3년 내 공공기관 정보화 사업 수행 실적 1건 이상
- 자본금 3억원 이상
- 컨소시엄 구성 허용 (최대 3개사)

## 5. 기술 요구사항

### 5.1 기능 요구사항 (FUR)

| ID | 요구사항 | 우선순위 |
|---|---|---|
| FR-01 | 실내 위치 추정 정확도 ±3m 이내 | 필수 |
| FR-02 | 출발지→목적지 경로 안내 (보행 동선) | 필수 |
| FR-03 | 다국어 4종 지원 | 필수 |
| FR-04 | 음성 안내 (TTS) | 권장 |
| FR-05 | 항공편 정보 연동 (게이트 변경 알림) | 권장 |
| FR-06 | 환승 동선 최적 추천 | 권장 |

### 5.2 비기능 요구사항

- 동시 사용자 5,000명 부하 처리
- 평균 응답 시간 1초 이내
- 가용성 99.5% 이상
- 개인정보 비식별 처리

### 5.3 표준 준수

- 전자정부 표준 프레임워크 준수
- 웹 접근성 (KS X OISO/IEC 40500)
- KISA 모바일 앱 보안 점검 통과
- 항공보안법 관련 데이터 처리 기준
