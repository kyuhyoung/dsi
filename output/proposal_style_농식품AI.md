## 4. 농식품AI 본문 구조 특수 패턴

### 4.1 평가 기준 ↔ 응답 위치 1:1 대응 (사업 전용)

B 공고 평가기준 ↔ D 응답 위치 매핑:
- B "기술 차별성·혁신성" → D 1-2
- B "상용화 준비도" → D 3-1, 3-3
- B "시장진입 전략" → D 3-2
- B "위기 대응" → D 3-5
- B "사업 실적" → D 4-5

(자세한 매핑은 `learned_patterns_농식품AI.yaml` 의 `rfp_response_mapping` 참조.)

### 4.2 글로벌 사업 강조 (해외시장 중심) — *주의 사항*

농식품AI 공고는 본디 *국내 신시장 창출이 주목적*이나, D는 **인도네시아·말레이시아 등 동남아 시장 중심**으로 작성됨:

- D 465~478행 "글로벌 진출 실적" — 4개국 수출 실적
- D 521~528행 "해외 전시회 4건 이상" — 국민체감활동을 해외 전시회로 채움
- D 524행 "국민체감활동으로 계획된 해외 전시회는 당사 자체 예산을 통해 추진할 예정임" — 국고 미사용 명시

→ **농식품AI 공고의 "국내 산업 대응력 우선" (B 97행) 대비 응답이 약함** — *보완 필요*.

### 4.3 짧은 문단·긴 단락 혼용 (D 특수)

- 본문 대부분 `*` 세부 설명은 1~2문장 단순 구조 (공통 규칙)
- 단, *6-1 추진계획* 부분(D 543~545행, D 555~558행)은 들여쓰기 없는 **장문 단락 서술** — 기술 배경 설명용
- 즉, **"기술 배경 = 장문 단락" / "성과·역량 = 3-tier 개조식"** 구분

## 5. 농식품AI 명시 어려운 영역

다음 항목들은 4개 문서로부터 명확히 추출하기 어려움:

1. **분량 가이드라인** — C 양식은 "분량 제한 없음, 필요시 이해를 돕기 위한 사진 등 삽입 가능" (C 95행)만 명시. D 실제 분량(텍스트 약 600행 + 50표 + 30그림)이 적정 기준인지 평가자 관점 불명
2. **표·그림 디자인 양식** — 텍스트 추출본은 `<표>`·`<그림>` 태그만 보유. 실제 hwp 원본의 색상·폰트·레이아웃 정보 부재
3. **평가위원이 가중치 부여하는 키워드** — B 평가기준 표가 텍스트 추출본에서 빈 `<표>` 로만 표시되어 세부 배점 항목 불명
4. **선정 사례 vs 탈락 사례 비교** — D 1건만 분석 가능, 선정 결과·평가자 코멘트 없음
5. **컨소시엄 4-4 섹션 응답** — D는 컨소시엄 헤더 명시되지 않음 (단독 신청 추정). 컨소시엄 사업의 응답 패턴 학습 불가
6. **양식 외 추가 콘텐츠 비중** — B 109행 "양식 외 추가 내용 가능" 명시. D가 양식 외 어떤 콘텐츠를 어디에 얼마나 추가했는지 정량 분석 어려움
