## 4. 시각 차이의 *시스템 원인 분석*

### 4-1. 원인 1: 양식 C용 .docx template 미등록 → *표지·셀 fill·헤더 스타일* 부재

- 현재 `templates/` 폴더에는 `dabeeo_proposal/` PPT template만 있음 (`ls templates/` 결과). **농식품AI 양식 C 전용 .docx template이 없음**.

### 4-2. 원인 2: `proposal-writer.md` 에이전트에 *표지/간지 명시 규칙 부재*

- `agents/proposal-writer.md` (확인 필요)는 "양식 C의 별지를 순서대로 채워라" 수준의 지시만 있을 가능성. D는 양식 C의 표 순서를 *재배열*하여 "별지 제3호 사업계획서 표지"를 1쪽으로, 자가진단서를 후반에 배치. 이 *큐레이션 룰*이 에이전트 프롬프트에 없음.

### 4-3. 원인 3: 그림 placeholder 시스템 부재

- D⁵는 본문 어디에도 `![그림1: 어스아이 플랜테이션 UI 캡처](path)` 같은 placeholder가 없음. 결과적으로 .docx 변환 시 *시각 자산 삽입 지점*이 제로.

### 4-4. 원인 4: 머리말·꼬리말·페이지 번호 export 설정 누락
