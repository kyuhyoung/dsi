# 다비오 주요 프로젝트·고객 실적

다비오는 모빌리티·공간 인프라·국토관리·국방·임농업 전 분야에서 대표 고객사 사업을 수행해 왔다.

## 모빌리티·디지털지도 (공간·인프라)

### 카카오모빌리티 / 카카오지도 — 구독 기반 지도 데이터 API 공급

- **사업 내용**: 기 구축한 실내지도 데이터를 API 형태로 카카오지도와 연동
- **공급 범위**: 국내 메이저 유통 3사를 포함한 주요 쇼핑몰·백화점·아울렛 등 **전국 100여 개 이상의 실내지도 데이터 제공**
- **모델**: 모빌리티 데이터를 API로 제공해 모빌리티 기업과 연동, 구독 기반 데이터 공급 모델로 확장
- **시기**: 2025년 카카오모빌리티 데이터 사업

### 티맵모빌리티 (TMAP) — 차세대 HDMAP 본사업

- **시기**: 2025년
- **내용**: 차세대 HDMAP 본사업 계약

### 네이버

- **사업 내용**: 국내 1·2위 맵 플랫폼(네이버·카카오) 대상 EARTHEYE STUDIO 서비스 공급
- **의의**: 국내 최대 커머셜 사이트 공간정보 관리 플랫폼 구축

## 국토관리·변화탐지 (공간·인프라)

### 대전시 — 변화탐지 정기 모니터링 사업

- **사업 내용**: 대전시 항공사진에 다비오 이미지 분석 AI 기술을 적용하여 건축물 변화를 분석
- **사업 기간**: **2021년 ~ 2025년** (매년)
- **분석 대상 지역**: 대전시 전체
- **의의**: 수십 년 반복하던 수작업 변화 탐지 판독 사업을 인공지능 기반으로 **3년 이상 지속 수행**하며 도시 내 건축물 변화를 정기적으로 분석·관리하는 모니터링 사업 수행

### 한국자산관리공사 (KAMCO) — 국유지 관리
