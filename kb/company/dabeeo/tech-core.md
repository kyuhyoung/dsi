# 다비오 6대 핵심 AI 백본 기술

다비오는 다중센서·다출처 영상 데이터를 처리하는 **6개 핵심 AI 백본 기술(The 6 Core AI Backbone Technologies)**을 보유하고 있다. 이 백본 기술들은 **Eartheye 플랫폼**의 기반이 된다.

## 6대 백본 기술 개요

| # | 기술명 (한글) | 기술명 (영문) | 핵심 내용 |
|---|---|---|---|
| 1 | 초해상화 | Super Resolution | 다출처 영상의 공간 해상도 품질을 최대 4배까지 향상시키는 영상 생성 기술 |
| 2 | 이미지 정합 | Image Alignment | 다중 영상 간 기하·지리적 오차를 정밀하게 보정하여 정합성을 확보하는 기술 |
| 3 | 의미론적 분할 | Semantic Segmentation | 지형·지물을 픽셀 단위로 정의된 클래스에 분류하고 세분화하는 기술 |

| 6 | 3차원 재건 | 3D Reconstruction | 중첩 다출처 영상데이터로 고정밀 3차원 공간을 복원 및 모델링하는 기술 |

## 기술별 상세

### 1. 초해상화 (Super Resolution)

- **최대 4배 해상도 향상**을 구현하여 원시 데이터 활용의 제약을 해소하고 정밀 분석 환경을 제공
- 대규모 공공 위성데이터의 활용 가능성을 확장하고, 고해상도 중심의 비용 문제를 해결하며, 국가 단위 분석 사업 경쟁력을 강화

### 2. 이미지 정합 (Image Alignment)

- 다중 영상 간 지리적 정합성을 고정밀로 확보하여 영상 분석의 정확도 극대화
- 변화 탐지 등 AI 분석 고도화의 필수 기반을 제공하는 핵심 전처리 기술

### 3. 의미론적 분할 (Semantic Segmentation)

- 멀티 해상도 영상에서도 강건하게 객체를 분류하여 공간데이터를 정밀하게 현행화
- 대규모 공간 데이터 구축·갱신의 효율성과 정확도를 동시에 높이는 핵심 기술

### 4. 객체 탐지 (Object Detection)

- **EO·SAR 등 다출처 영상 분석**으로 정밀한 마이크로 객체 탐지 및 분류 수행
- 광범위 지역의 정밀 모니터링이 필요한 국방·감시·정찰, 임업·농업 등에서 고도화된 버티컬 솔루션으로 확장
- **적용 사례**: 팜나무 객체 탐지, 구상나무 고사목 객체 탐지, SAR 선박 객체 탐지·분류

### 5. 변화 탐지 (Change Detection)

- 국토 지형(토지·산림·수계) 환경 변화부터 객체 단위의 도시 환경 변화(건물·도로·폐기물)에 이르기까지 **총 16개 유형의 변화 탐지** 가능
- **국내 최고 수준의 변화 탐지 서비스** 구현
- **적용 사례**: 건물 변화 탐지, 토지 변화 탐지

### 6. 3차원 재건 (3D Reconstruction)

- 위성·항공 영상을 기반으로 정밀하고 경쟁력 있는 LoD(Level of Detail)를 달성
- 객체 단위 분리가 가능한 3D 모델로 높은 활용성과 호환성을 제공하며 다양한 환경 조건에서도 강건한 3차원 재건 기술 연구개발

**LoD 달성 수준**:
- 위성: LoD 1.3 이상
- 항공: LoD 2.2 이상

**3차원 공간 재건 4단계 프로세스**:

1. **진정사 영상 구현** — 진정사 영상 추출 (TOP) + 빌딩 마스크 데이터 구축
2. **3D 포인트 클라우드 추출**
3. **3D 메시 모델 생성** — 세그멘테이션 + 메시 단순화 및 용량 최적화
4. **고정밀 3차원 공간 재건** → Phase 4 Spatial OS

## Eartheye 플랫폼 시리즈 (버티컬 라인업)

다비오 원천 기술 플랫폼 **Eartheye**를 기반으로 버티컬 영역별 Product-Market Fit을 검증한 AI-Powered 도메인 특화 플랫폼 **'Eartheye' 버티컬 라인업** 확보.

### 5대 버티컬 영역

1. 복합시설·인프라
2. 국토 정보·관리
3. 임업·농업
4. 국방·감시·정찰
5. 공간·학습

### 주요 제품 플랫폼

#### EARTHEYE LABELBOX (공간·학습)

공간정보 특화. 상황인지 정밀 학습데이터를 생성·관리하기 위한 **AI-오토 라벨링 기반의 학습데이터 생성 플랫폼**.

- 다비오 인공지능과 연계하는 오토라벨링 지원 / AI 객체탐지 및 세그멘테이션 연계용 자동 라벨링 지원
- 이미지 및 동영상 모두 지원, 라벨링 기능 지원 / 객체 트래킹 기능으로 타겟 객체를 따라가면서 라벨링 진행
- 작업자 관리 및 교차검증 / 생산과 데이터 검증(검수) 작업 구분, 교차검증 지원
- **시장 검증 완료**: 3백만 개 이상 데이터 제작 / **국방과학연구소, 한국자산관리공사 데이터 사업** 사용

#### EARTHEYE STUDIO (공간·인프라)

160개 이상의 API를 통해 확장성 있는 **실내외 통합 디지털 트윈 플랫폼**.

- 상용화 API 및 데이터 제공 플랫폼 / **160여 개의 Developer API** 제공
- LoD 3-4 수준의 3D 데이터 생성 및 편집 지원 / 인공지능 기반 베이스 폴리곤 자동 생성
- 디지털 트윈레벨(3D)의 실내외 통합 정보 제공 / 지오레퍼런싱 및 노드링크 연결로 실내외 Pathfinding 지원
- **국내 최대 커머셜 사이트 공간정보 관리 플랫폼** / **네이버, 카카오 등 국내 1·2위 맵 플랫폼** 대상 서비스 공급
- 전국민을 End-User로 하는 국내 최대 모빌리티 플랫폼 기업들과의 협업으로 데이터 구독시장 선점

#### EARTHEYE CHANGES (국토 정보·관리)

다출처/다중 영상 기반 AI 분석으로 **광범위 지역 변화를 신속·정밀 탐색·분석**.

- 관심 객체 탐지 및 영역 변화 탐지 / 특정 타겟 영역의 변화를 실시간 분석, 영역 상황인지
- Polygon 기반 데이터 제공 (3rd Party 서비스 연동) / 건물·도로·지형 정보 등 공간 데이터를 Polygon 정보 제공
- 특화 UX 지원으로 고객 경험 증대 (Intuitive UX) / End-user 업무 편의성 및 생산성을 높이는 UX/UI
- **대한민국 전국토 모니터링 및 국방 감시정찰 서비스** / **대한민국 최초 전국토 모니터링 및 분석 서비스** 사용

#### EARTHEYE PLANTATION (임·농업)

임농업 버티컬 시장 선점을 위한 **플랜테이션 특화 다출처 영상 AI 분석 솔루션**.

- **세계 최초 30cm 위성 기반 팜(오일) 농장 관리 솔루션** / 3,000sqkm 이상 면적의 상용 분석 서비스로 고객 검증 완료
- 초정밀 분석 인공지능기술 지원 (세계 최고 수준) / **99% 이상의 탐지 정확도, 95% 이상의 식생 건강도 분석**
- 농장(영역) 토지매력도 3차원 분석 정보 제공
- 웹·모바일 플랫폼 연동 통합 솔루션 / 시장 검증·상용화 완료

#### EARTHEYE ISR (국방·감시·정찰)

**국내 최초 초소형위성체계 SAR 영상의 AI 기반 융합 분석**을 통해 해양 모니터링 기술 개발.

- 해양 선박탐지 특화 기능 지원 / AIS 융합지원, 10종 선박탐지 및 Heading 정보 표시
- SAR 특화 선박 탐지 및 분류 인공지능 기술 / **선박탐지 F1 Score 90%, 10종 선박분류 성능 80% 초과 달성**
- **초고속(5분) 분석처리 기술 제공** / 5분 단위 초소형위성 SAR 영상에 대한 고속 분석 기술
- **국내 최초 40기 초소형위성 SAR 정보분석 시스템** / 국가 감시 사업을 위한 해양 탐지 서비스 운영 예정

## 시장 문제 해결 효과 (3대 가치)

상황인지 분석 자동화로 시장의 비효율 'Bottle-Neck'을 근본적으로 해결.

| 가치 | 효과 |
|---|---|
| **시간 단축** | 광범위한 분석의 반복적·수동 작업을 단축 |
| **비용 절약** | 인공지능 기술을 통해 자동화 체계 / 데이터 생성 프로세스 효율화 |
| **지속 가능** | 자동화에 따른 효율성 극대화 |

**As-Is** (수작업): 영역 크기에 따라 수많은 인력 필요 / 전 과정을 수작업으로 진행
**To-Be** (AI 자동화): 인공지능 기술을 통해 자동화 체계 / 데이터 생성 프로세스 효율화
