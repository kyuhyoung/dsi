# 항공전자·레이더 사업

## 항공전자 정비·생산 대상 기종

당사가 항전장비 정비·창정비·구성품 정비 등을 수행하는 주요 기종:

| 기종 | 운용 |
|---|---|
| KT/A-1 | 훈련기 |
| T/A-50 | 훈련기·경공격기 |
| KF-16 | 전투기 |
| F-15K | 전투기 |
| KF-X (KF-21) | 차세대 전투기 |
| E-X | 공중조기경보 |

## 주요 항전장비 영역

- **MC** (Mission Computer, 임무 컴퓨터)
- **MFD** (Multi-Function Display, 다기능 디스플레이)
- **FLCC** (Flight Control Computer, 비행제어 컴퓨터)
- **HUD** (Head-Up Display, 전방시현장치)

## 레이더 사업

| 영역 | 내용 |
|---|---|
| 저고도 레이다 | 정비, 창정비 |
| 항공관제 레이다 | 정비, 창정비 |
| 레이더 구성품 | 정비 및 수리부속 생산 |

## 본 RFP 적용성 (예시)

- 항공기 통합 시험평가 경험
- 항전장비 다년 정비·창정비 노하우
- 다양한 기종 동시 지원 능력
