# 당사 정밀유도무기 라인업

당사는 대한민국에서 제작하는 **대부분의 유도무기**를 개발·생산합니다.

## 주요 체계

| 무기체계 | 영역 | 상태 |
|---|---|---|
| **천궁-II (M-SAM)** | 중거리 지대공 | 양산·수출 |
| **현궁** | 보병용 휴대 대전차 미사일 | 양산 |
| **신궁** | 휴대용 지대공 미사일 | 양산 |
| **비궁** | 유도로켓 시스템 | 차세대 |
| **해궁** | 함대공 유도무기 | 양산·수출 (말레이) |
| **K-SAAM** | 함대공 미사일 | 양산 |
| **백상어** | 잠수함 발사 어뢰 | 양산 |
| **홍상어** | 함정 발사 어뢰 | 양산 |

## 통합 역량의 의미

이 라인업은 단순 제품 다양성이 아니라 다음을 의미:

1. **다중 플랫폼 적용 경험** — 지·공·해 전 플랫폼 통합
2. **양산 라인 다수 운영** — 품질·형상관리 표준의 검증
3. **수출 트랙레코드** — 글로벌 시장 진입 채널
4. **시험평가 도메인 깊이** — 각 무기체계의 시험·인수 누적 경험

## 본 RFP 적용성 (예시)

- 정밀유도무기 양산 실적 요건 (다수 체계 동시 충족)
- 시험평가 도메인 깊이 (본 정책 연구의 기반)
- 통합 무기체계 사업 경험
