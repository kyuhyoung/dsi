# 현궁 — 보병용 휴대 대전차 미사일

## 체계 개요

- **명칭:** 현궁 (보병용 중거리 유도무기)
- **분류:** 정밀타격무기체계 (PGM) — 대전차 미사일
- **개발 주관:** 국방과학연구소(ADD) 전체 개발
- **양산:** LIG D&A (유도탄 체계 개발·생산), 한화 (발사대)

## 개발 일정

- 2007년 ADD 주관 개발 착수
- 2015년 개발 완료
- 연구개발비 1,500억원 투입
- 2016년 생산 시작 → 육군 전방·해병대 배치

## 주요 사양

| 항목 | 값 |
|---|---|
| 사거리 | 2.5 km |
| 관통력 | 900mm 장갑 (2.5km 거리) |
| 무게 | 13.0 kg (미사일 + 발사관) |
| 가격 | 미사일 1발당 1억원 초과 |

## 양산 현황

- 향후 7년간 양산 사업 규모: **약 1조원**
- **2024-12-24: 4차 양산 계약 1,002억원** (방위사업청)
- 육군·해병대 단계적 공급
