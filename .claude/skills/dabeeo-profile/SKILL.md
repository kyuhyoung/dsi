4. 전자전 (EW)

## 사용 지침 (proposal-writer 등 agent용)

1. **본 skill = 인덱스·요약** — 회사 핵심 수치 빠른 참조용
2. **상세 자료 = `kb/` 폴더** — 평가배점·요건에 맞는 자료를 Grep으로 검색
3. **인용 시 출처 명시** — KB 파일 경로 (예: *"(kb/projects/2024-사우디천궁-II수출.md)"*)
4. **격식체 사용** — `proposal-korean-style` skill 참조
5. **데모 단계 한정** — 실제 입찰 제출 금지

차후 자료가 들어오면:

| 자료 종류 | 저장 위치 | 파일명 예시 |
|---|---|---|

| 새 프로젝트 실적 | `kb/projects/` | `2026-XX사업.md` |
| 새 기술 자료 | `kb/tech/` | `<기술명>.md` |
| 과거 제안서 | `kb/proposals/` | `<RFP번호>/proposal.md` |

각 파일 상단에 frontmatter 메타데이터(`title`, `period`, `client`, `keywords`, `source`) 필수.

`Glob "kb/**/*.md"`와 `Grep`만으로 자동 활용됩니다. 본 skill 수정 불필요 (단, 새 폴더가 생기면 위 KB 라우팅 표에 한 줄 추가 권장).
