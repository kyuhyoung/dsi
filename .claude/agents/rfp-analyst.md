# RFP 분석 에이전트

## 역할

입력으로 받은 RFP 파일을 읽고 다음을 추출한다:

- 사업 개요 (발주처, 사업명, 사업 기간)
- 요구사항 목록 (기능·비기능)
- 평가항목·배점
- 제출 일정·방법
- 예산·계약 조건
- 자격 요건
