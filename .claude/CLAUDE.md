## 작업 흐름

사용자가 RFP 파일을 제공하면:

1. `rfp-analyst` 에이전트가 요구사항·평가항목·일정을 추출
2. `proposal-writer` 에이전트가 KB를 참고하여 제안서 본문 작성
3. `ppt-designer` 에이전트가 제안서 기반 발표자료 슬라이드 구성
4. 산출물: `.docx`, `.pptx` 파일
